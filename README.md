# 🚀 Space Shooter

A classic **Console-based Space Shooter Game** built with **C++** using the Windows Console API.

This project was developed as my **first university project** to practice programming fundamentals, object-oriented programming, and basic game development concepts.

---


## ✨ Features

- 🚀 Player-controlled spaceship
- 👾 Three different enemy types
- 💥 Bullet collision & explosion effects
- ❤️ Health (HP) System
- 📈 Score & Level progression
- ⚡ Dynamic difficulty scaling
- 🎯 Three difficulty modes
  - Easy
  - Medium
  - Hard
- ⏸ Pause / Resume system
- 🎨 Colored ASCII graphics
- 🖥 Smooth console gameplay
- 📍 Centered game window

---

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬅ Left Arrow | Move Left |
| ➡ Right Arrow | Move Right |
| Space | Shoot |
| P | Pause / Resume |
| Q | Quit Game |

---

## 👾 Enemy Types

| Enemy | HP | Score |
|--------|---:|------:|
| `\V/` Basic | 1 | +50 |
| `>X<` Fast | 1 | +100 |
| `[W]` Tank | 3 | +150 |

## ▶ Getting Started

### Requirements

- Windows OS
- g++ (MinGW) or Microsoft Visual Studio

### Compile

```bash
g++ main.cpp -o SpaceShooter
```

### Run

```bash
SpaceShooter.exe
```

---

## 🎯 Gameplay

- Destroy enemy spaceships.
- Avoid enemy bullets.
- Earn points by defeating enemies.
- Survive as long as possible.
- Higher levels increase enemy speed and spawn rate.
